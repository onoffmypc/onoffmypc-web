Flash your ESP32

  1. Download the firmware
    Clone or download onoffmypc-firmware-release and open it in VS Code with the PlatformIO extension.
  2. Create src/config.h
    Copy src/config.example.h to src/config.h and fill in your values: 
    #define WIFI_SSID     "YourWiFi"
#define WIFI_PASSWORD "YourPassword"
#define DEVICE_ID     "..."   // from your dashboard
#define DEVICE_TOKEN  "..."   // the token you just copied
#define PC_MAC        "AA:BB:CC:DD:EE:FF"  // your PC's MAC
  3. Wire up the ESP32
    Connect DHT22 to pin D4 · status LED to D2 · power sense to D34 · power button to D26 · reset button to D27.
  4. Flash and monitor
    pio run -e esp32dev --target upload && pio device monitor
    The LED will blink slowly while connecting, then go solid when online.
