Set up your ESP32

  1. Flash the firmware
    Download the latest .bin from the firmware releases page and flash it to your ESP32 via USB.
  2. Connect to the setup network
    On first boot the device creates a Wi-Fi network named OnOffMyPC-XXXXXX. Connect your phone or laptop to it — a setup page should open automatically. If it doesn't, open a browser and go to http://192.168.4.1.
  3. Enter your details
    Fill in your Wi-Fi name and password, then paste the Device ID and Token from the previous screen. Add your PC's MAC address for Wake-on-LAN (run ipconfig /all on Windows or ip link on Linux/macOS). Tap Save & Connect.
  4. Wire up the ESP32
    DHT22 → D4 · LED → D2 · power sense → D34 · power button → D26 · reset button → D27. Enable Wake-on-LAN in your BIOS and network adapter settings.

The LED blinks while connecting and goes solid when online. To reconfigure, send r over the serial monitor.
